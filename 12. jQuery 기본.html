jQuery 기본

  • 제이쿼리는 자바스크립트로 만들어진 라이브러리(프레임워크)
  • 라이브러리(jQuery....)는 함수(특정기능)를 만들어놓은 것
  • 프레임워크(vue, react, angular..)는 작업환경을 제공하는 것
  • 버전 1.xx(구 브라우저와 호환), 2.xx(구버전x, 가볍게 이용하기 위함),3.xxx(기능강화, 구버전x)
  • minified : 압축버전, slim(간소화버전, 애니메이션+ajax 등 몇 가지를 제외한 버전)
  • https://releases.jquery.com/

주요 메서드

  • css() : style 속성을 변경, 가져올 때
  • attr() / removeAttr() : 요소의 속성을 변경, 가져올 때 / 속성을 지울 때
  • html() : html로 가져오거나 대입
  • text() : text로 가져오거나 대입
  • addClass(), removeClass(), toggleClass() : 클래스 지정 메서드들

수치관련 메서드들

  • width()/height() : 폭, 높이값을 적용/가져올 때
  • offset() : 절대좌표값을 반환, offset().left
  • scrollTop() : 스크롤 높이값을 가져오거나 지정

객체(DOM) 편집 메서드

  • before() : 선택자 앞에 내용 추가
  • after() : 선택자 뒤에 내용 추가
  • append() : 선택자 자식의 맨뒤로 추가
  • prepend() : 선택자 자식의 맨앞으로 추가
  • clone() : 선택한 요소를 복사
  • remove() : 선택한 요소를 삭제
  • empty() : 선택자 자식을 모두 지움

기타 메서드

  • parent() : 선택자 부모를 선택
  • children("li") : 선택자 자식을 선택
  • not("") : 괄호 안을 제외한 나머지 선택
  • next() : 선택자 다음 형제를 선택
  • prev() : 선택자 이전 형제를 선택
  • hasClass("on") : 선택자에 클래스가 있으면 true, 없으면 false값 반환
  • val() : value 속성값을 얻거나 지정
  • is(":checked") : 선택자에 checked 속성이 있으면 true, 없으면 false, :selected / :visible / :hidden
  • get(0) : 선택자를 자바스크립트 선택자로 변환
  • each() : 선택자 갯수만큼 반복할 때 유용
