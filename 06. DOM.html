DOM(Document Object Model)

  • HTML 문서의 구조와 그 정보를 말함.
  • 선택자.명령/속성
  • 아이디 선택자: getElementById()
  • 클래스 선택자: getElementsByClassName
  • 요소 선택자 : getElementsByTagName()
  • css 선택자 : querySelector("css선택자")
  • 복수 css 선택자 : querySelectorAll("css선택자")
  • 폼선택자 : document.("formName")
  • 선택자.style.속성 : css 속성을 얻거나 설정
  • 선택자.태그의 속성명 : 요소의 속성값을 얻거나 설정
  • 선택자.innerText : 글자내용을 얻거나 바꿀때
