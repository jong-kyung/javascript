이벤트와 애니메이션

이벤트

  • load() : 선택자의 자손을 포함한 외부소스가 로딩이 완료된 후 이벤트 실행
  • ready() : 선택자의 html 요소만 로딩 후 이벤트 실행
  • 이벤트 (렌더링)순서 : DOM 생성 > ready() > 렌더링(콘텐츠표현) > load()
  • click() / dblclick()
  • mouseout() / mouseover() : 마우스가 벗어났을 때, 마우스가 호버했을 때
  • mousemove() / mousedown() : 마우스가 움직일 때, 마우스가 클릭을 했을 때(keydown과 비슷 / 누르자마자 실행)
  • focus() / blur() : 포커스를 얻었을 때, 잃었을 때
  • keypress() / keydown() : 키눌렀을 때(특수키x), 대문자만(특수키o)
  • keyup() : 키를 (눌렀다) 떼었을 때
  • scroll() : 스크롤 했을 때
  • change() : 값이 변경 되었을 때
  • on() : 이벤트리스너(이벤트 감시자), 여러개의 이벤트를 걸거나 동적요소에 이벤트를 걸때 : $(선택자).on(이벤트종류,이벤트대상,함수정의)

효과 및 애니메이션

  • hide() : 선택자를 안보이게 숨김
  • show() : 선택자를 보이게 함
  • toggle() : 토글(숨김/보임)
  • fadeIn() : 점점 보이게
  • fadeOut() : 점점 안보이게
  • fadeToggle() : 부드럽게 토글
  • slideUp() : 위로 숨김
  • slideDown() : 위로 보임
  • slideToggle() : 위/아래로 토글
  • animate() : 애니메이션 적용
