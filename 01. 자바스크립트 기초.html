자바스크립트 기본

자바스크립트의 특징

  • 웹의 3요소 : 구조(html), 디자인(css), 동작(Jacascript)
  • 브라우저에서 실행되는 세계에서 가장 많이 사용되는 언어
  • 자바(백엔드)와 다른 클라이언트(프론트엔드) 언어
  • 컴파일 과정이 없는 인터프리터 언어
  • 소스가 노출, 원본에 대한 저작권 권리 행사가 힘듦.
  • 플랫폼/운영체제 제한이 적음
  • Pure script, 순수스크립트, 바닐라스크립트는 라이브러리(jQuery)를 사용하지 않는 자바스크립트를 말함
  • 라이브러리(명령어/함수를 미리 만들어 놓은 것), 프레임워크(작업환경을 제공, veu/angular/react....)
  • 대소문자를 구분
  • 실행문을 종료할때는 세미콜론(;) 적용
  • 큰따옴표(")와 작은따옴표(') 겹침주의
  • 모든 괄호 (), {}, []는 열고 닫는 원칙
  • 주석 : /* 여러줄 주석 */, // 말꼬리주석, 한줄주석

자바스크립트 작성방법

  • 인라인 스크립트 : 비권고
  • 내부스크립트: script 요소(태그)로 head/body 안에 사용
  • 외부스크립트: script 내용을 외부파일로 불러오는 방식
  • 스크립트 위치는 /body 전이나 head에는 defer 속성을 권장

변수

  • 변수란 변하는 수, 값, 데이터, 문자, 불리언(Boolean)값
  • 선언은 var 선언문, let, const
  • 변수의 데이터형 : 문자, 숫자, 논리형, null, undefined
  • 변수 네이밍 규칙
    - 첫글자는 숫자가 올 수 없다.
    - 특수기호, 빈칸, 한글은 안되고, 영문과 숫자로 조합
    - 기본적으로 소문자만 사용(이유있는 경우 제외)
    - 의미에 맞는 이름 사용
    - 언더바표기법(snakecase) : super_man
    - 낙타표기법(camelcase) : superMan
    - 같은 이름으로 중복 선언 x, 예약어(특정명령어, 컴퓨터와 약속된 단어...)와 같은 단어 사용불가
