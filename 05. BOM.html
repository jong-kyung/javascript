BOM(Browser Object Model)

  • 브라우저에서 제공하는 객체모델
  • window, screen, location, history, navigator
  • window: open(), alert(),prompt(), confirm(), moveTo(), resizeTo(), setInterval() 반복실행, setTimeout() 지연실행
  • screen: width, height, colorDepth
  • location: href, hash, hostname, protocol
  • history: back(), forward()
  • navigator: appCodeName, appName, language, product, platform, userAgent
